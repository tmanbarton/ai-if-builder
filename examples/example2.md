This game is called "The Lighthouse". The player is a lighthouse keeper doing their evening rounds.

### Locations:
- Shore path
- Lighthouse base
- Storage room
- Spiral stairs
- Lamp room

### Map
- Shore path, north -> lighthouse base
- Lighthouse base, south -> shore path
- Lighthouse base, east -> storage room
- Lighthouse base, up -> spiral stairs
- Storage room, west -> lighthouse base
- Spiral stairs, down -> lighthouse base
- Spiral stairs, up -> lamp room
- Lamp room, down -> spiral stairs

### Items:
- Matches - storage room
- Oil can - storage room
- Logbook - lighthouse base
